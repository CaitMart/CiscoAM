# CiscoAM
gather part numbers and serial numbers from cisco devices into a unified format. 

There is the issue of collecting Cisco part numbers and serial numbers. These will later be added to compliance documentation for asset management purposes. In order automate the process and get rid of excessive copy and paste of .txt to excel sheet, automation will occur for this long drawn out process.

The script will open the .txt file and go to the inventory section. From there it will save the necessary part and serial numbers. After the script reads all necessary data, it will then export it to an excel sheet. This will allow for documentation of all devices that are being utilized and will make for an easier process when adding it to compliance documentation.
